### Vector Shortcuts

| Short-Hand | Code |
|:----:|----|
| `vi` | `vector<int> a;` |
| `vf` | `vector<float> a;` |
| `vc` | `vector<char> a;` |
| `vvi` | `vector<vector<int>> a;` |
| `vvf` | `vector<vector<float>> a;` |
| `vvc` | `vector<vector<char>> a;` |
